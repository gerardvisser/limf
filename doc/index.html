int bsf (void)
Bit scan forward. Returns the position of the least significant bit set, or 0 if there are no bits set.
Examples:
if this integer has value 0, this function returns 0;
if this integer has value 1, this function returns 0;
if this integer has value 2, this function returns 1;
if this integer has value 3, this function returns 0;
if this integer has value 4, this function returns 2;
if this integer has value 8, this function returns 3.
int bsr (void)
Bit scan reverse. Returns the position of the most significant bit set plus one, or 0 if there are no bits set.
Examples:
if this integer has value 0, this function returns 0;
if this integer has value 1, this function returns 1;
if this integer has value 2, this function returns 2;
if this integer has value 3, this function returns 2;
if this integer has value 4, this function returns 3;
if this integer has value 8, this function returns 4.

Adds the absolute value of other to the absolute value of this integer. The absAdd function returns true if the result of the addition is too large to be represented by this integer. The sign of this integer does not change.
bool absDec (void)
Decreases the absolute value of this integer by one. If the result is zero, the sign will always be false. This function always returns false.
Warning: The absDec function may never be called when the magnitude of this integer is zero.
bool absInc (void)
Increases the absolute value of this integer by one. If the result is too large to be represented by this integer, its value becomes 0 and true is returned. The sign of this integer does not change.
bool absSub (Integer* other)
Subtracts the absolute value of other from the absolute value of this integer. The sign of this integer changes if the absolute value of other was larger than the absolute value of this integer. The sign of this integer becomes false if both absolute values were equal. The absSub function always returns false.
